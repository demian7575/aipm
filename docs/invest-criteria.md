# INVEST Criteria

## Definition
**I**ndependent  
**N**egotiable  
**V**aluable  
**E**stimable  
**S**mall  
**T**estable  

## Criteria Details

### Independent
- Can be developed without dependencies on other stories
- Doesn't block or require other stories to be complete
- Has clear boundaries

### Negotiable
- Details can be discussed and refined
- Not overly prescriptive about implementation
- Allows for creative solutions

### Valuable
- Provides clear business or user benefit
- Stakeholders understand the value
- Justifies development effort

### Estimable
- Team can size the effort required
- Complexity is understood
- Has clear scope boundaries

### Small
- Fits within one iteration/sprint
- Can be completed by small team
- Not too complex to manage

### Testable
- Has clear acceptance criteria
- Success/failure can be verified
- Measurable outcomes defined

## Validation in AIPM
- System checks INVEST compliance automatically
- ChatGPT provides AI-powered analysis when configured
- Warnings can be overridden with confirmation
- Stories must pass INVEST to reach "Ready" status
