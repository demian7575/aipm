# Display GitHub PR Link in Development Tasks Card

## Story Details
When a user creates a pull request for a user story, the Development Tasks card should display a clickable link to the GitHub PR, enabling quick navigation to review code changes and track development progress. The link should only appear after PR creation and include visual indicators for PR status (open/merged/closed).

## Acceptance Criteria
Title: Display GitHub PR Link in Development Tasks Card

As a: project manager
I want: to see a direct link to the GitHub pull request in the Development Tasks card
So that: I can quickly navigate to review code changes and track development progress without searching for the PR manually
